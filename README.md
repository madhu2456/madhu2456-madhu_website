# 🌐 Personal Portfolio – madhudadi.in

A modern, responsive portfolio website built with Next.js to showcase my work, projects, and experience in data analytics, backend systems, and digital transformation.

---

## 🚀 Live Website

- 🔗 https://madhudadi.in

---

## 🧠 About the Project

This portfolio serves as a central hub to:

- Showcase my professional experience and projects  
- Highlight technical skills in analytics, backend, and digital systems  
- Share work across domains like data science, marketing analytics, and system design  
- Provide an interactive and clean user experience  

---

## ⚙️ Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript / JavaScript  
- **Styling:** Tailwind CSS  
- **Deployment:** Vercel / Custom Hosting  
- **Analytics:** Google Tag Manager (GTM) + GA4  

---

## ✨ Features

- 📱 Fully responsive design  
- ⚡ Fast performance with Next.js optimization  
- 🎯 Clean UI/UX for better readability  
- 📊 Integrated analytics tracking (GTM + GA4)  
- 🔗 Project showcase with live links  
- 🌐 Custom domain setup
